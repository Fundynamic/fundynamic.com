Test Driven Development

Writing tests is vital for agility. Writing tests first is even better. Driving your design through tests results in code that complies to YAGNI, DRY and more.

In this workshop I give you a hands-on experience. I'll briefly tell you what TDD is and then we practice using several kata's.
